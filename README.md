# Rakuten Workspace

楽天アフィリエイト運用の親ワークスペースです。商品選定、投稿候補、ブログ公開物を管理します。

## 起動方法

この親フォルダ自体には起動コマンドはありません。ブログは `rakuten-affiliate-blog/` を参照します。

## フォルダ構成

- `rakuten-affiliate-blog/`: 静的HTMLの楽天ブログ
- `.codex-remote-attachments/`: Codex添付ファイル
- `archive/`: 古いログ、退避資料

## 主要機能

- 楽天ブログ公開物の管理
- 商品選定・ROOM投稿文作成の作業拠点
- 公開済み商品の重複管理
