# Current Spec

## Overview

Static HTML Rakuten affiliate blog deployed to Vercel.

## Files

- `index.html`: homepage
- `about.html`: about page
- `privacy.html`: privacy page
- `articles/`: article pages
- `assets/`: styles and images
- `data/`: product management data

## Verification

- Open locally through `serve.bat` or a static server.
- Check affiliate links, PR labels, and article display before publishing.
