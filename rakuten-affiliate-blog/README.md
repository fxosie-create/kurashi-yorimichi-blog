# Rakuten Affiliate Blog

静的HTMLの楽天アフィリエイトブログです。暮らし系の記事、商品紹介、PR表記、Vercel公開を扱います。

## 起動方法

```bat
serve.bat
```

または静的ファイルとして `index.html` を確認します。

## フォルダ構成

- `index.html`: トップページ
- `about.html`: サイト説明
- `privacy.html`: プライバシー
- `articles/`: 記事HTML
- `assets/`: CSS・画像
- `data/`: 公開済み商品などの管理データ
- `archive/`: 旧README、過去ログ、退避資料

## 主要機能

- 楽天アフィリエイト記事の公開
- PR/広告表記
- 公開済み商品の管理
- Vercel公開
